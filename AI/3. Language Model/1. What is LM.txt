"""
    언어 모델(Language Model, LM)
        - 언어라는 현상을 모델링하고자 단어 시퀸스(또는 문장)에 확률을 할당(assign)하는 모델
        - 통계 이용, 인경 신경망 이용으로 나뉨.
        
        - 단어 시퀸스에 확률을 할당하기 위해서 보편적으로 '이전 단어들이 주어졌을 때 다음 단어를 예측'
        - 그 외에는 주어진 양쪽의 단어들로부터 가운데 비어있는 단어를 예측하는 언어 모델

    언어 모델의 평가
        - 펄플렉서티(Perplexity, PPL)
            (1) PPL은 수치가 낮을수록 언어 모델의 성능이 좋다는 것을 의미
            (2) 단어의 수로 정규화된 테스트 데이터에 대한 확률의 역수
            (3) 언어 모델이 특정 시점에서 평균적으로 몇 개의 선택지를 가지고 고민하는지를 의미하는 수치
            (4) PPL은 테스트 데이터에 의존하므로 두 개 이상의 언어 모델을 비교할 때는 정량적으로 양이 많고, 또한 도메인에 알맞은 동일한 테스트 데이터를 사용해야 신뢰도가 높다.
"""
